Overview

This class provides a simpler way to replicate the often-used pattern of a searchable Core Data-backed table view. Must be used as a subclass.

fetchRequestForSearch:

Returns a new NSFetchRequest for the specified search string.

- (NSFetchRequest *)fetchRequestForSearch:(NSString *)searchString

Parameters

searchString

The query entered by a user. May be nil.

Return Value

The NSFetchRequest to be executed by the NSFetchedResultsController.

Discussion

Returns a new NSFetchRequest for the specified search string.

Override this method in your subclass to return the appropriate NSFetchRequest for the search term. If searchString is nil, return your unfiltered dataset.

Warning: This method must be overidden in your subclass.
